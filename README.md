# Application to demonstrate Customer Job Details React App when we click on specific user

Components Used:
1) ReactJs
2) node package manager
3) React Router 

# Things to Install:

## Install Nodejs 
Goto https://nodejs.org/en/download/ and download

## Create React App 
Run 
```bash
npm install -g create-react-app
```
to download create-react-app tool to create React app easily
and then run
```bash
npm create-react-app app-name
```
app-name here is customer-job-details-react

## Steps to start application
Go to folder where package.json is present. And then Run

Run
```bash
npm install
```
It will download all the modules required for the project

Run
```bash
npm start
```

## Components Used
**Customers** : Main Customer Component

**CustomerDescription**  : This is child component of above Customer Component

## EndPoints

To get the list of Customers in details exposing it on /listCustomers
